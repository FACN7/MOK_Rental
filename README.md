### `how to install`

You can clone the repo by typing the command

```console
> git clone https://github.com/nataphilips/emn-week5-facn7.git
> cd week3-NIM-todo-list
> npm i
> google-chrome index.html
```
### `how to test`

Install tape and tap-spec

```console
> npm i tape -D
> npm i tap-spec -D
```

Run the test command

```console
> npm test
```
